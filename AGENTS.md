# Agent Instructions

Todo agente de IA que trabalhar neste repositório deve seguir este fluxo antes de começar qualquer tarefa:

1. Ler [`.agents/README.md`](.agents/README.md).
2. Ler [`.agents/memory/project-memory.md`](.agents/memory/project-memory.md).
3. Ler [`.agents/memory/active-context.md`](.agents/memory/active-context.md).

Ao terminar uma tarefa relevante:

1. Atualizar `project-memory.md` se descobriu fatos estáveis sobre o projeto.
2. Atualizar `active-context.md` com estado operacional recente, decisões, riscos ou pendências.
3. Manter as anotações curtas, factuais e úteis para o próximo agente.

Regras:

- Não apagar memória útil sem substituir por versão melhor.
- Não registrar segredos, tokens ou credenciais.
- Não transformar esses arquivos em log verborrágico de terminal.
- Priorizar fatos de arquitetura, deploy, caminhos importantes, convenções e decisões recentes.
